## 安装/使用说明

``` bash
# 克隆项目
git clone git@github.com:tyyenplus/shop-mall.git

# 安装依赖
npm install

# 本地开发环境，运行demo 访问:http://localhost:8080
npm run dev

# 打包文件
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
